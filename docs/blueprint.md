# **App Name**: Campus Hub

## Core Features:

- Secure Authentication: Firebase Authentication with email/password, admin-only account creation, and password reset functionality.
- Role-Based Access Control: Admin and student roles with restricted access to data and functionalities based on custom claims.
- Results Management: Admin interface to upload and edit student academic results, with student view for their individual results.
- Fees Management: Admin tool to set and update student fee status (Paid, Pending, Partial), and student view to check their fee status.
- Announcement System: Admin tool to create and manage announcements targeted at all students or specific classes, with a feed displayed on student dashboards.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to represent knowledge and sophistication.
- Background color: Light gray (#EEEEEE), a desaturated version of the primary color to create a clean and modern feel.
- Accent color: Blue (#2196F3) for interactive elements and important notifications to convey trust and clarity.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality; suitable for both headlines and body text.
- Use clear and simple icons from a consistent set (e.g., Material Icons) for navigation and actions.
- Employ a clean, card-based layout with clear information hierarchy, spacing, and alignment.
- Use subtle transitions and animations to enhance user experience and provide feedback on interactions.
