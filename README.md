# Campus Hub: Modern Student Portal

Campus Hub is a comprehensive campus management system built with Next.js 15, Firebase, and Genkit. It provides a robust interface for administrators to manage students, academic results, and fees, while offering students a personalized dashboard to track their progress.

## ðŸš€ Key Features

### For Administrators
- **Student Management:** Create, edit, and delete student accounts.
- **Academic Results:** Upload and manage results manually or via bulk Excel/XLSX upload.
- **Fee Management:** Track payment status (Paid, Partial, Pending) with automatic balance calculation.
- **Announcement System:** Post targeted announcements to specific classes or the entire student body.
- **Site Content Manager:** Edit the public landing page text and images directly from the dashboard.
- **Activity Log:** A chronological audit trail of system activities.

### For Students
- **Personalized Dashboard:** View relevant announcements.
- **Academic Reports:** Access beautiful, printable report cards for different terms and sessions.
- **Fee Tracker:** Monitor outstanding balances and payment history.

### Environment Variables
To run this project, you need to set up the following environment variables (though client-side config is hardcoded for the prototype):

- `FIREBASE_SERVICE_ACCOUNT_JSON`: The full JSON object of your Firebase Service Account for Admin SDK operations.

### Local Development
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
